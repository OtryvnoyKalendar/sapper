создать в скачанном проекте папки build и cmakebin

установить mingw с sourseforge.
в нём нужно выбрать все пакеты кроме fortran, ada, objc

прописать в переменную среды Path (у вас может быть свой путь) C:\MinGW\bin\

установить cmake с официального сайта

скачать с github релиз sdl. Нужен sdl2. Я использовал SDL2-devel-2.26.5-mingw.
В папке i686-w64-mingw32 есть всё необходимое для компиляции нашей игры с sdl2. Использую 32 бит для большей совместимости
из папки i686-w64-mingw32\bin нужно переместить sdl2.dll в cmakebin. Если переместить dll в другое место, то программа не запуститься
из папки i686-w64-mingw32\lib нужно переместить все файлы кроме папок в C:\MinGW\lib (ну или куда вы там mingw скачали)
из папки i686-w64-mingw32\include нужно перемесить папку SDL2 в папку C:\MinGW\include

библиотеки opengl32 и glu32 есть по дефолту в MinGW. с ними ничего делать не нужно

дальше открыаем папку проекта в powershell
запустите скрипт компиляции прописав в консоль: .\project_start.bat

приложение должно скомпилироваться в папку cmakebin
